# 🎮 Pong en Python

Mon premier jeu développé en Python.

## 📖 Description

Ce projet est un jeu réalisé dans le cadre de mon apprentissage de Python en atelier périscolaire de calsse de 4ème. Le joueur doit faire rebondire une balle sur les côté de l'écran tout en l'empêchant de sortir du cadre. Comme Pong, le jeu emblèmatique des années 1970.

Ce projet m'a permis de découvrir :

- Les bases de Python
- les variables (globales/définies)
- Les boucles et conditions (if, for, while)
- La logique de jeu

## 🚀 Fonctionnalités

- ✅ déplacement de la racket
- ✅ Système de score  

## 🛠️ Technologies utilisées

- Python 3.x
- Bibliothèque(s) : pygame, asyncio
